# swagdown
`swagdown` is a Golang library and CLI for generating markdown documentation from JSON or YAML Swagger files. 

> WIP (2016-10-04)

### Usage

#### CLI

```bash
$ go get github.com/patrickrand/swagdown

$ swagdown <path>/<to>/swagger.json 

```
